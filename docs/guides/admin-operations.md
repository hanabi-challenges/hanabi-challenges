# Admin Operations Guide

## Core Admin Areas

- Admin home
- Events index (including drafts)
- Users
- CMS/content editor
- Badge sets

## Event Management

- Publish/unpublish controls
- Edit and delete controls
- Owner/delegate control surface for leagues

## User Management

- Role updates (admin/superadmin policies)
- Manual role repair only with clear audit justification

## Content Operations

- Edit About/FAQ/Legal content through CMS
- Verify markdown rendering in both themes

## Notifications

- Badge awards and request approvals should generate notifications.
- Ensure actionable requests clear from queues once resolved.

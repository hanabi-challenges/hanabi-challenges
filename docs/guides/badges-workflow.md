# Badge Design Workflow

## Designer Flow

1. Open badge designer
2. Choose shape
3. Enter icon token (Google Material icon)
4. Configure set-level main/secondary text
5. Configure per-tier inclusion/size
6. Save / Save As (unique set name required)

## Persistence Expectations

Saved set includes:

- Builder config
- Consolidated SVG preview

## Event Attachment

Attachment is pull-based from event wizard/event admin surfaces.
Designer should not push attachment state.

## Award Display

- User badge grids support mixed 1x1 and 2x2 layout classes.
- Badge modal should remain theme-aligned.

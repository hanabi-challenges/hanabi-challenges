# Event Workflows

## Event Creation

1. Open admin events index
2. Start wizard
3. Select event type
4. Fill event info
5. Optional badge attachment step (type-dependent)
6. Publish via wizard action area

## League Workflow

1. Create league event
2. Configure/manage queued games in admin rail
3. Set live toggle
4. Start game (handles active-game conflict confirmation)
5. Ready-check window runs
6. Team assignment + team page handoff
7. Score submit returns players to event page
8. Session/event end finalizes ongoing games

## Challenge Workflow

1. Publish challenge
2. Users play/submit by seed rules
3. Stats and standings available via tabs

## Tournament Workflow

1. Publish bracket/tournament
2. Participation closed and elimination-based visibility rules apply

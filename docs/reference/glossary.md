# Glossary

- **League**: UI name for `session_ladder` event format.
- **Ready Check**: short confirmation window before team assignment/start.
- **Queued Game**: league game configured but not started.
- **In-Progress Game**: active league round currently being played/scored.
- **Completed Game**: finalized league round with recorded results.
- **Owner**: admin user who created the event (`owner_user_id`).
- **Delegate**: user granted event management rights by owner/superadmin.
- **Badge Set**: reusable design + tier configuration for awards.
- **Tier**: badge rank variant (gold/silver/bronze/participant).
